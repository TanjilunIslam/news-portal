Difference between var let and const:

var declares a variable scoped to the function it's declared in, or globally if declared outside a function. It can be re-declared and reassigned.

Difference between arrow function and a regular function:

A regular function is defined with the function keyword and has its own this context. An arrow function, however, doesn't bind its own this, arguments, super, or new.target - they inherit them from the containing scope.

Difference between map, forEach, filter and find:

map transforms each element of an array and returns a new array with the same length. forEach executes a provided function once for each array element. filter creates a new array with elements that pass a test defined by a provided function. find returns the first element in an array that satisfies a provided testing function.
